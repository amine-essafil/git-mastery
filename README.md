# git-mastery

A hands-on project to master Git step by step.

The goal is simple:
learn Git by actually using it — writing code, making commits,
fixing mistakes, and documenting the process.

This repo follows a structured path from basic commands
to real-world team workflows.

## 🧱 Project Structure

Each module is organized like this:

- `README.md` → explains the goal and key commands  
- `exercises/` → practical tasks to apply concepts  
- `GITLOG.md` → personal notes and lessons learned  

---

## 🎯 Approach

- Practice Git through real changes (not just theory)  
- Make small, meaningful commits  
- Learn from mistakes and fix them  
- Keep track of progress in each module
